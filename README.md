# TerminEngine

TerminEngine is a console-based set of Python classes
used to produce ASCII games.

Color Features:

    Full Color
    Alpha Blending

Event Features:

    Live Keypress (Linux)
    Single Keypress (Linux)

Time Features:

    Timer
    Clock
    FPS Lock

Additional Features:

    Full 2D Vector Class
    Font Class
    Seven Segment Display Class
    Sprite Class
    Drawing Class
    Bitmask Class
    Bit Map Class
